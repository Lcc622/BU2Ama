# 加色加码系统修复计划

基于 Codex 深度分析，确定以下修复优先级和方案。

## Phase 1：修复 DA/DM 不匹配和后缀生成（最高优先级）

### 问题 1.1：DA 文件无法加载
**根因**：`excel_processor.py:24-25` 正则只匹配 `EP|DM|PZ`，不匹配 `DA`

**影响**：`DA-0.xlsm` 等文件在自动扫描时被排除

**修复位置**：`backend/app/core/excel_processor.py`

**修复方案**：
```python
# 行 24-25：修改正则接受 DA
STORE_PREFIX_PATTERN = re.compile(r"^(EP|DM|DA|PZ)-", re.IGNORECASE)
STORE_SOURCE_FILE_PATTERN = re.compile(r"^(EP|DM|DA|PZ)-\d", re.IGNORECASE)

# 添加归一化函数（参考 follow_sell_processor.py:38）
def _normalize_store_prefix(self, store_prefix: Optional[str]) -> str:
    normalized = str(store_prefix or "").strip().upper()
    if normalized == "DA":
        normalized = "DM"
    return normalized if normalized in STORE_PRIORITY else "EP"
```

### 问题 1.2：后缀生成逻辑错误
**根因**：`follow_sell_processor.py:279` 从现有 listing 复制后缀，而不是根据店铺+尺码规则生成

**影响**：生成的 SKU 后缀不符合业务规则

**修复位置**：`backend/app/core/follow_sell_processor.py`

**修复方案**：
```python
# 添加店铺后缀生成函数
def _generate_suffix_for_store(self, store_prefix: str, size: str) -> str:
    """根据店铺和尺码生成后缀"""
    try:
        size_num = int(re.search(r'\d+', str(size)).group())
    except:
        size_num = 0

    is_plus = size_num >= 14

    normalized_store = self._normalize_store_prefix(store_prefix)
    if normalized_store == "DM":  # DAMA
        return "-PLPH" if is_plus else "-PL"
    elif normalized_store == "PZ":
        return "-DAPH" if is_plus else "-DA"
    else:  # EP
        return "-PH" if is_plus else "-USA"

# 修改 find_sizes_for_skc() 中的 SKU 生成逻辑
# 不再从 SELECT DISTINCT size, suffix，改为只查 size
# 然后调用 _generate_suffix_for_store() 生成后缀
```

## Phase 2：统一后缀处理（高优先级）

### 问题 2.1：excel_processor 中的 EP 硬编码
**位置**：
- `excel_processor.py:1885` - 默认后缀 `-USA`
- `excel_processor.py:1212` - 后备主题后缀
- `excel_processor.py:1558` - `suffix_candidates()` 只处理 `-USA/-PH`

**修复方案**：
```python
# 添加统一的后缀 helper
def _get_store_suffix_family(self, template_type: str) -> Tuple[str, str]:
    """返回 (普通后缀, Plus后缀)"""
    if "DAMA" in template_type or "DM" in template_type:
        return ("-PL", "-PLPH")
    elif "PZ" in template_type:
        return ("-DA", "-DAPH")
    else:  # EP
        return ("-USA", "-PH")

def _is_plus_body_suffix(self, suffix: str) -> bool:
    """判断是否为 Plus body 后缀"""
    return suffix.upper() in {"-PH", "-PLPH", "-DAPH"}

def _generate_suffix(self, template_type: str, size: str) -> str:
    """根据模板类型和尺码生成后缀"""
    try:
        size_num = int(re.search(r'\d+', str(size)).group())
    except:
        size_num = 0

    base_suffix, plus_suffix = self._get_store_suffix_family(template_type)
    return plus_suffix if size_num >= 14 else base_suffix

# 修改所有硬编码位置使用这些 helper
```

### 问题 2.2：suffix_candidates() 不支持 DAMA/PZ
**位置**：`excel_processor.py:1561`

**修复方案**：
```python
def suffix_candidates(suffix: str, template_type: str) -> List[str]:
    suffix = (suffix or "").strip().upper()
    if not suffix:
        base_suffix, plus_suffix = self._get_store_suffix_family(template_type)
        return [base_suffix, plus_suffix, ""]

    # 返回同族后缀变体
    base_suffix, plus_suffix = self._get_store_suffix_family(template_type)
    if suffix == base_suffix:
        return [base_suffix, plus_suffix, ""]
    elif suffix == plus_suffix:
        return [plus_suffix, base_suffix, ""]
    else:
        return [suffix, ""]
```
