## 📝 特殊字段说明

### 1. 固定值字段

以下字段通常使用固定值，不需要从输入表映射：

| 输出表字段 | 固定值 | 说明 |
|-----------|--------|------|
| Update Delete | "Update" | 表示更新操作 |
| Relationship Type | "Variation" | 表示变体关系 |

### 2. 计算字段

以下字段需要通过计算或转换生成：

| 输出表字段 | 生成规则 | 说明 |
|-----------|---------|------|
| Seller SKU | 从输入表 SKU 转换 | 颜色/尺码变更 |
| Style Number | 从 SKU 提取 | 去掉颜色、尺码、后缀 |
| Parent SKU | 从 SKU 生成 | 产品代码 + 后缀 |
| Launch Date | 根据北京时间计算 | 3PM 规则 |
| Your Price | 从 All Listing Report 匹配 | 根据 SKU 匹配价格 |
| Colour Map | 从颜色代码映射 | 使用颜色分类 |

### 3. 图片 URL 字段

图片 URL 需要根据产品代码和颜色代码生成：

```
https://eppic.s3.amazonaws.com/{产品代码}{颜色代码}-{变体}.jpg
```

**示例**：
- Main Image URL: `EG02088MG-L1.jpg`
- Other Image URL: `EG02088MG-L2.jpg`, `EG02088MG-L3.jpg`, ...

## 🔄 跨后缀加色的特殊处理

在跨后缀加色场景中（例如 EG02088-PH → EG02084-USC），某些字段的映射规则需要特殊处理：

| 输出表字段 | 数据来源 | 说明 |
|-----------|---------|------|
| Seller SKU | 原产品 style + 新颜色 + 目标后缀 | EG02088MG14-USC |
| Parent SKU | 目标产品 Parent | EG02084-USC |
| Product Name | 目标产品样本数据 | 从 EG02084-USC 的样本复制 |
| Product Description | 目标产品样本数据 | 从 EG02084-USC 的样本复制 |
| Key Product Features | 目标产品样本数据 | 从 EG02084-USC 的样本复制 |
| Main Image URL | 原产品 style + 新颜色 | EG02088MG-L1.jpg |
| Your Price | 原产品价格 | 从原产品的 All Listing Report 获取 |

## 📚 使用建议

### 1. 优先使用高置信度映射

48个高置信度映射可以直接使用，这些字段的映射关系非常明确。

### 2. 验证中等置信度映射

36个中等置信度映射需要根据实际数据验证，特别是：
- 价格相关字段
- Occasion Lifestyle 系列字段
- 尺寸测量字段

### 3. 人工确认特殊字段

以下字段需要人工确认或使用特殊逻辑：
- Update Delete（固定值）
- Relationship Type（固定值）
- Style Number（从 SKU 提取）
- Your Price（从 All Listing Report 获取）
- Launch Date（根据时间计算）

### 4. 实现建议

```python
# 伪代码示例
def map_fields(input_row, mappings):
    output_row = {}

    # 1. 使用高置信度映射
    for mapping in high_confidence_mappings:
        output_field = mapping['output_field']
        input_field = mapping['input_field']
        output_row[output_field] = input_row[input_field]

    # 2. 处理固定值字段
    output_row['Update Delete'] = 'Update'
    output_row['Relationship Type'] = 'Variation'

    # 3. 处理计算字段
    output_row['Seller SKU'] = generate_sku(input_row['SKU'], new_color, new_size)
    output_row['Style Number'] = extract_style(input_row['SKU'])
    output_row['Launch Date'] = calculate_launch_date()

    # 4. 处理价格字段
    output_row['Your Price'] = get_price_from_listing_report(input_row['SKU'])

    # 5. 处理图片 URL
    output_row['Main Image URL'] = generate_image_url(style, color, 'L1')

    return output_row
```
