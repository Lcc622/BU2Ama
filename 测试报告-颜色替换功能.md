## 🔍 技术实现细节

### 后端实现
- **语言**: Python 3
- **框架**: FastAPI
- **Excel 处理**: openpyxl
- **颜色映射**: 114 个颜色代码映射表

### 前端实现
- **框架**: React 18 + TypeScript
- **构建工具**: Vite
- **状态管理**: Zustand
- **UI 组件**: Tailwind CSS

### 核心算法
1. **SKU 解析**: 正则表达式提取产品代码、颜色代码、尺码、后缀
2. **颜色替换**: 字符串替换 + 颜色映射表查询
3. **图片 URL 生成**: 模板字符串 + 颜色代码替换
4. **价格计算**: 原价 + 10 美元
5. **Launch Date**: UTC 时间 + 8 小时（北京时间）+ 3PM 规则
