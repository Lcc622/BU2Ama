## ✅ 已完成的修复

### 1. 修复价格逻辑 ✅

**问题**：
- Your Price 错误地 +10
- 缺少 List Price, Business Price, Quantity Price 计算

**修复前**：
```python
base_price = price_map.get(info.sku, 79.99)
new_price = base_price + 10  # ❌ 错误
ws.cell(row=row_idx, column=cols["price"] + 1, value=new_price)
```

**修复后**：
```python
# Your Price (standard_price) - 直接使用，不加10
your_price = price_map.get(info.sku, 79.99)
ws.cell(row=row_idx, column=cols["price"] + 1, value=your_price)

# List Price = Your Price + 10
if "list_price" in cols:
    list_price = your_price + 10
    ws.cell(row=row_idx, column=cols["list_price"] + 1, value=list_price)

# Business Price = Your Price - 1
if "business_price" in cols:
    business_price = your_price - 1
    ws.cell(row=row_idx, column=cols["business_price"] + 1, value=business_price)

    # Quantity Price = Business Price * 0.95
    if "quantity_price" in cols:
        ws.cell(row=row_idx, column=cols["quantity_price"] + 1, value=business_price * 0.95)

    # Quantity Price 2 = Business Price * 0.92
    if "quantity_price_2" in cols:
        ws.cell(row=row_idx, column=cols["quantity_price_2"] + 1, value=business_price * 0.92)

    # Quantity Price 3 = Business Price * 0.90
    if "quantity_price_3" in cols:
        ws.cell(row=row_idx, column=cols["quantity_price_3"] + 1, value=business_price * 0.90)
```

**修复位置**：`excel_processor.py` 行 339-368

### 3. 完善图片 URL 规则 ✅

**问题**：缺少 PH 后缀的特殊处理

**钉钉要求**：
- EP店铺 + PH后缀：L101-105
- EP店铺 + 非PH后缀：L1-5
- PZ/DAMA店铺 + PH后缀：PL101-105
- PZ/DAMA店铺 + 非PH后缀：PL1-5

**修复前**：
```python
# 只有简单的 L1-5 或 PL1-5
main_image_url = f"...{image_variant}1.jpg"
other_image_urls = [f"...{image_variant}{i}.jpg" for i in range(2, 7)]
```

**修复后**：
```python
# 根据后缀判断图片格式
if suffix == "-PH":
    # PH 后缀使用 101-105 格式
    main_image_url = f"...{image_variant}101.jpg"
    other_image_urls = [
        f"...{image_variant}102.jpg",
        f"...{image_variant}103.jpg",
        f"...{image_variant}104.jpg",
        f"...{image_variant}105.jpg",
        f"...{image_variant}106.jpg"
    ]
else:
    # 非 PH 后缀使用 1-5 格式
    main_image_url = f"...{image_variant}1.jpg"
    other_image_urls = [
        f"...{image_variant}2.jpg",
        f"...{image_variant}3.jpg",
        f"...{image_variant}4.jpg",
        f"...{image_variant}5.jpg",
        f"...{image_variant}6.jpg"
    ]
```

**修复位置**：`excel_processor.py` 行 383-408
